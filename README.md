# 北京邮电大学数电随堂实验代码

这里是23级老登的数电随堂实验的代码。希望能帮你们节省一些垃圾时间！

虽然我的数电没考好，但祝大家数电都能高分！

说回来，数电在某些方面确实很有意思，但也要大家遇到好老师才行呀TAT。
